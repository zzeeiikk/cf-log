Nutzungsbedingungen

Allgemeines

cf-log ist eine kostenlose Webanwendung zur Dokumentation von Trainingsdaten. Die Nutzung erfolgt auf eigene Verantwortung.

Verfügbarkeit

Wir bemühen uns um eine hohe Verfügbarkeit der Anwendung, können jedoch keine Garantie für eine ununterbrochene Nutzbarkeit geben. Wartungsarbeiten können zu zeitweiligen Ausfällen führen.

Datenspeicherung

cf-log bietet verschiedene Speichermethoden an. Die Verantwortung für die Sicherheit und Verfügbarkeit der Daten liegt bei der gewählten Speichermethode:

  • GitHub Gists: Unterliegt den Nutzungsbedingungen von GitHub
  • WebDAV: Unterliegt den Bedingungen Ihres WebDAV-Anbieters
  • Supabase: Unterliegt den Nutzungsbedingungen von Supabase

Haftungsbeschränkung

Wir haften nicht für Datenverluste, die durch technische Probleme, Benutzerfehler oder Probleme bei externen Diensten entstehen. Wir empfehlen regelmäßige Backups Ihrer Trainingsdaten.
