Erste Schritte

1

App öffnen

Öffne cf-log in deinem Browser unter app.html oder klicke auf "Zur App" in der Navigation.

Tipp: Für die beste Erfahrung empfehlen wir Chrome, Firefox oder Safari.

2

Speichermethode wählen

Wähle eine Speichermethode aus: GitHub Gists (kostenlos) oder WebDAV (eigener Server).

Empfehlung für Anfänger: Starte mit GitHub Gists - das ist kostenlos und einfach einzurichten.

3

Erstes Training eintragen

Trage dein erstes Training ein. Du kannst Übungen hinzufügen, Sätze und Wiederholungen dokumentieren.

Los geht's: Klicke auf "Neues Training" und beginne mit deiner ersten Übung!

Speicher einrichten

GitHub Gists (Kostenlos)

1

Erstelle einen kostenlosen GitHub-Account unter github.com

2

Gehe zu den Einstellungen und erstelle ein Personal Access Token

3

Gib deinen GitHub-Benutzernamen und das Token in cf-log ein

Vorteile: Kostenlos, automatische Backups, Versionierung, von überall zugänglich

WebDAV (Eigener Server)

1

Richte einen WebDAV-Server ein (Nextcloud, ownCloud, NAS-System)

2

Erstelle einen Ordner für deine Trainingsdaten

3

Gib die WebDAV-URL, Benutzername und Passwort in cf-log ein

Vorteile: Volle Kontrolle, unbegrenzter Speicherplatz, eigene Infrastruktur

Training dokumentieren

Neues Training erstellen

1. Klicke auf "Neues Training"

2. Wähle das Datum (Standard: heute)

3. Füge einen Titel hinzu (z.B. "Push-Training")

4. Klicke auf "Training erstellen"

Übungen hinzufügen

1. Klicke auf "Übung hinzufügen"

2. Wähle eine Übung aus der Liste oder erstelle eine neue

3. Trage Sätze, Wiederholungen und Gewichte ein

4. Füge Notizen hinzu (optional)

5. Klicke auf "Speichern"

Training bearbeiten

• Klicke auf ein Training in der Liste, um es zu bearbeiten

• Du kannst Übungen hinzufügen, löschen oder bearbeiten

• Änderungen werden automatisch gespeichert

Statistiken nutzen

Übungsstatistiken

  • • Gewichtsverlauf über Zeit
  • • Persönliche Bestleistungen
  • • Trainingshäufigkeit
  • • Volumen-Analysen

Allgemeine Statistiken

  • • Trainingskalender
  • • Wöchentliche/monatliche Übersichten
  • • Trainingsdauer
  • • Übungsverteilung

Daten exportieren

Du kannst deine Trainingsdaten als CSV-Datei exportieren:

1. Gehe zu den Einstellungen

2. Klicke auf "Daten exportieren"

3. Wähle den Zeitraum aus

4. Die CSV-Datei wird automatisch heruntergeladen

Mobile Nutzung

Als App installieren

1

Öffne cf-log in Chrome oder Safari auf deinem Handy

2

Tippe auf das Teilen-Symbol (Safari) oder das Menü (Chrome)

3

Wähle "Zum Startbildschirm hinzufügen"

Offline-Nutzung

cf-log funktioniert auch offline:

  • • Trainings eintragen ohne Internetverbindung
  • • Daten werden lokal gespeichert
  • • Automatische Synchronisation bei Internetverbindung
  • • Keine Datenverluste

Probleme lösen

Häufige Probleme

App lädt nicht

• Überprüfe deine Internetverbindung

• Lade die Seite neu (F5)

• Lösche Browser-Cache

Synchronisation funktioniert nicht

• Überprüfe deine Speicher-Einstellungen

• Stelle sicher, dass du online bist

• Versuche es später erneut

Daten verschwunden

• Überprüfe deine Speicher-Einstellungen

• Stelle sicher, dass du mit dem richtigen Account verbunden bist

• Kontaktiere uns bei GitHub Gists-Problemen

Browser-Kompatibilität

Empfohlen: Chrome, Firefox, Safari, Edge (neueste Versionen)

Nicht unterstützt: Internet Explorer

PWA: Chrome, Safari (iOS), Edge
